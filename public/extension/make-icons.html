Avena icon generator

Chrome Web Store requires PNG icons at 16, 48, and 128 px. This page renders them from icon.svg via canvas, then downloads the PNGs so you can drop them back into public/extension/.

Click each button to download. Zip the full folder to submit the extension.

After you have the three PNG files in this folder, run:

cd public/extension
zip -r extension.zip . -x "make-icons.html" "icon.svg"
